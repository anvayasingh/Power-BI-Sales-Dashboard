# BuyNex Sales Analysis Report (Power BI Dashboard)

This project presents an interactive year-wise sales analysis for BuyNex, designed to uncover insights into business performance and customer trends. The dashboard leverages data modeling and visualization in Power BI to provide an in-depth view of sales metrics across regions, customer segments, and time periods.

# Key Features & Components

Comprehensive Data Model: Integrated multiple tables, including Dates, Orders, Order Status, Order Type, Payment Type, Region, State, and Town, for dynamic filtering and drill-down analysis.

Performance KPIs: Female & Male Customers, Gross Cost, Profit, Revenue, Order Count, and Sales YoY (Year-over-Year) growth.

Visual Insights: Highlighted the Top 5 Regions by sales and profit, enabling performance benchmarking and strategic decision-making.

Time-Series Analysis: Year-wise and monthly comparisons for sales and profitability trends.

Interactive Dashboard: Custom slicers and filters for state, region, and order type analysis.

# Tech Stack

Tool: Power BI Desktop

Data Source: Tabular data model with relational tables (Orders, Payments, Geography)

Skills Demonstrated: Data modeling, DAX measures, KPI creation, and interactive visualization
